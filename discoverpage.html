Question-3: What is event delegation in the DOM?


Event delegation attaches a single event listener to a parent element to handle events on its child elements via event bubbling. It reduces the need for multiple listeners by detecting events on descendants through a unified handler.

Question-4: What is event bubbling in the DOM?


Event bubbling is a propagation method where an event starts from the target element and moves up through its ancestors in the DOM.
